# COOKIE SNIFFER

### INSTALLATION
```
=> Search for Cookie sniffer in Chrome Webstore
```
OR
```
=> Clone this repository
=> Enter chrome://extensions in your Google Chrome address bar
=> Click on `Load unpacked extension...` option and select the cloned directory
```

### SETUP
```
=> After installation a cookie icon appears on the top right corner
=> Clicking on it will present you with an input form
=> Enter your ACHE server credentials
   Eg: http://localhost:8080 (The URL should not have a trailing / character and must have http/https protocol)
   NOTE: This will not persist the DDT server info
=> DDT server credentials will automatically be captured provided the service is running on PORT 8084
```
